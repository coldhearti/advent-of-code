## Installation
To install the package run:
~~~bash
pip install .
~~~

To uninstall the package run:
~~~bash
pip uninstall aoc2022
~~~

## To run:
Navigate to the package folder and run:
~~~bash
python main.py
~~~
Solutions print to CLI.
